💊 Smart Medicine Dispenser

The Smart Medicine Dispenser is an Arduino-based automated system designed to help elderly people and patients who often forget to take their medicines on time.
The system automatically dispenses medicine, alerts the user using sound and light, and sends SMS notifications to caregivers for better monitoring.

🚀 This project was developed during Semester II as part of Integrated Project – I (IP-I). It was built by a team of four, focusing on solving a real-world healthcare problem using embedded systems and basic electronics.

✨ Key Features

Automatic medicine dispensing using a servo motor

Multiple compartments for different medicines

LCD display for system status and messages

Buzzer and LED alerts for reminders

Push-button acknowledgment after medicine pickup

SMS alerts using a GSM module
